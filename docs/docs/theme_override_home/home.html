Cutevariant

A standalone and free application to explore genetics variations from VCF file

Get started Download now

Features

wysiwyg Desktop Application

Cutevariant is a cross-platform desktop application used to explore genomics variation from a VCF files. It is Easy to install on any computer using Windows, Mac or Linux. Unlike other solutions, there is no need to send files to a third party. All your data stays with you. Exploring genomics variants is faster and you don't have the latency of a web application.

people Community

Cutevariant is a new opensource project written in Python and Qt framework and available on github. Contribution are welcome. You can create new plugin and share them with the community. Since it is a new project, bug report and any feedback will be highly appreciated.

code Import VCF files

Cutevariant works by importing VCF file into a sqlite database. It supports Snpeff and VEP annotation. You can create a new project and incrementally import VCF files. You can then make a database of all your variants and use it to query them later.

code VQL Language

Select and filters variants directly from a domain specific language that looks like a subset of SQL. It makes possible to create all kinds of query that you could not do with any other user interface. If you don't like query manually, you can always build them indirectly from plugins available from the mainwindow.

extension Plugins

Cutevariant is composed entirely of plugins. It means that it is easily customizable at will. You can create custom controllers, themes and support new file formats for importing or exporting. All plugins are made with Python.
